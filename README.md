# INDOGRAM CYK Parser

INDOGRAM CYK Parser adalah aplikasi web untuk melakukan parsing kalimat baku Bahasa Indonesia menggunakan Context-Free Grammar (CFG), Chomsky Normal Form (CNF), dan algoritma Cocke-Younger-Kasami (CYK).

Aplikasi ini dibuat sebagai tugas mata kuliah Teori Bahasa dan Otomata.

Link : https://indogram-cyk-parser-wjypwtq9xclymwmytz2fkx.streamlit.app/

## Fitur

- Memasukkan kalimat Bahasa Indonesia
- Mengecek apakah kalimat diterima atau ditolak
- Menggunakan aturan grammar CFG dan CNF
- Menampilkan hasil parsing menggunakan algoritma CYK
- Dibangun menggunakan Python dan Streamlit
